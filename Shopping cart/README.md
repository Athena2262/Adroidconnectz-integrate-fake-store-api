
# Build With

-   HTML5
-   CSS3
-   JavaScript
-   ReactJS with Hooks, Lifecycle Methods, and Router
-   Fake Store API

# Features

-   Image Slider at the HomePage
-   Wishlist Page
-   Men and Women Section
-   CheckOut Page

# Getting Started

```
git clone https://github.com/khunhour/shopping-cart.git
cd shopping-cart
npm install
npm start
```
